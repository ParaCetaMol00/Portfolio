About This Project

This is my personal developer portfolio ‚Äî designed and built completely from scratch without any templates or frameworks. The goal was to create something that feels uniquely mine: dark, sharp, and full of personality.

The biggest challenge was handling the animations and scroll interactions using pure vanilla JavaScript and CSS. No libraries. This forced me to understand how IntersectionObserver works, how to manage RAF loops for smooth cursor tracking, and how to orchestrate CSS keyframes precisely.

I also paid close attention to typography and spacing ‚Äî pairing Bebas Neue for display text with DM Sans for readability and JetBrains Mono for code-flavored accents. Every section has a reason for the font choice.

What I Learned

Building this taught me how much detail matters in UI ‚Äî the difference between a generic page and a polished one is often just 20 small decisions. I got much more comfortable with CSS custom properties, clip-path shapes, and layered backgrounds.
