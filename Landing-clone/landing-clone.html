About This Project

This project was a deliberate exercise in pixel-perfect reproduction. I picked a real-world SaaS landing page and rebuilt it from scratch ‚Äî no inspecting the source code, just visual matching from the browser.

The challenge was translating a design I didn't build into clean, structured HTML and CSS. This forced me to think carefully about layout systems, spacing consistency, and component decomposition ‚Äî skills that carry directly into any professional frontend role.

I paid special attention to cross-browser compatibility and tested the finished result on Chrome, Firefox, and Safari, making adjustments wherever rendering differed.

What I Learned

Cloning a design sharpens your eye dramatically. I started noticing subtle things ‚Äî shadow softness, button border-radius consistency, section padding rhythm ‚Äî that I would have overlooked before. It's one of the best ways to level up CSS skills fast.
